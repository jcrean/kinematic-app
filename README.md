# kinematic-app

A Leiningen template to generate a boilerplate Kinematic app.

## Usage

    $ lein new kinematic-app myapp
    $ cd myapp
    $ lein jar
    $ bash run-server.sh

Navigate to localhost:8080/myapp/api/hello where a sample api route is located.
